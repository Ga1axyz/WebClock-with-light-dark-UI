# WebClock-with-light-dark-UI
使用 HTML, CSS, JS 制作的时钟，支持明暗主题切换，与移动设备兼容

## 介绍
本项目基于 https://github.com/bedimcode/responsive-clock-ui ，在其基础上略微优化了样式与代码
